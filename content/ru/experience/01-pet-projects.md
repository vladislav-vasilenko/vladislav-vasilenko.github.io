- **TalkyBuddy** ([banibot.ru/talkybuddy](https://banibot.ru/talkybuddy)) — Full-stack AI экосистема обучения иностранным языкам с фокусом на голосовое взаимодействие в реальном времени.
    - Спроектировал **кроссплатформенную экосистему** (Web, iOS/Android, Telegram Mini App).
    - Разработал модульное ядро на основе **Domain-Driven Design (DDD)** для изоляции сложной лингвистической логики от инфраструктуры.
    - Внедрил **WebRTC и Omni-modal AI**: спроектировал систему обмена аудио с низкой задержкой для голосового ИИ, достигнув времени отклика менее секунды (**TTFT**).
    - **Оркестрация ИИ-агентов**: реализовал адаптивную систему сборки ИИ-тьюторов с циклами самокоррекции, использующую накопленный опыт для улучшения сценария урока.
    - *Стек*: React Native, Next.js, Node.js (NestJS), WebRTC, OpenAI/Gemini API.
- **BaniBaniBot** ([banibanibot.t.me](https://banibanibot.t.me)) — Платформа мультимодальной генерации визуального контента в Telegram.
    - Интегрировал несколько генеративных моделей (**OpenAI GPT-Image 1.5**, **Gemini 3.1 Flash / Gemini 3 Pro** native image generation) с единым API и автоматическим роутингом между провайдерами.
    - Реализовал пайплайн подготовки промптов с помощью **LLM** (Gemini 3.1 с Reasoning и Search Grounding) для повышения качества и контролируемости визуальной генерации.
    - Применил **Domain-Driven Design (DDD)** в архитектуре сервиса.
    - Спроектировал **Event-Driven Architecture (EDA)** в бэкенде для масштабирования обработки тяжелых задач инференса, поддержки сложных асинхронных пайплайнов (FSM).
    - Реализовал систему **Credits Ledger** для управления балансом и учета затрат на генерацию GPU вычислений.
