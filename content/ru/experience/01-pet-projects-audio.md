- **TalkyBuddy** ([banibot.ru/talkybuddy](https://banibot.ru/talkybuddy)) — Full-stack AI экосистема для обучения иностранным языкам с фокусом на голосовое взаимодействие в реальном времени.
    - Спроектировал **кроссплатформенную экосистему** (Web, iOS/Android, Telegram Mini App).
    - Внедрил **WebRTC и Omni-modal AI (Speech-to-Speech translation)**: спроектировал аудиопайплайн с низкой задержкой для голосового ИИ, интеграция Audio-LM / ASR / TTS-моделей (достигнуто время отклика TTFT < 1s).
    - Разработал модульное ядро на основе **Domain-Driven Design (DDD)** для изоляции сложной логики от инфраструктуры.
    - **Оркестрация ИИ-агентов**: реализовал адаптивную систему сборки ИИ-тьюторов.
    - *Стек*: React Native, Next.js, Node.js (NestJS), WebRTC, OpenAI/Gemini API, DSP ML.
- **BaniBaniBot** ([banibanibot.t.me](https://banibanibot.t.me)) — Платформа мультимодальной генерации контента в Telegram.
    - Интегрировал несколько генеративных моделей и мультимодальных LLM (Gemini / OpenAI API). Плотная работа с промптингом и контролируемой генерацией.
    - Спроектировал **Event-Driven Architecture (EDA)** в бэкенде для масштабирования обработки тяжелых задач инференса, поддержки асинхронных FSM (что аналогично пайплайнам обработки аудио).
    - MLOps & Billing: реализовал систему **Credits Ledger** для учета GPU вычислений.
